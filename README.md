# NBX Live Results Scraper

A simple web scraper that fetches virtual sports results from BetPawa Uganda and displays them in a minimal mobile-friendly interface. Built with Flask and Selenium.

## Running on Replit

1. Import this repository into Replit.
2. Replit will automatically install dependencies from `requirements.txt`.
3. The `replit.nix` file (if included) will ensure Chrome and chromedriver are available.
4. Click the "Run" button â€“ the app will start on port 8080.
5. Open the webview to see live results.

Note: The scraper runs on each page refresh. It may take a few seconds to load.
